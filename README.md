# Vid-organ - Video Organizer

Organize and rename video files in a folder with:
- **Automatic movie title and year extraction**
- **Interactive TMDb & IMDb verification** ✨ NEW!
- **Audio language metadata setting**
- **Smart file organization**

## ✨ New Feature: Interactive Verification

When organizing videos, the script now:
1. Extracts movie title and year from filename
2. **Verifies with TMDb and IMDb APIs**
3. **Shows close matches for user selection**
4. **Allows manual override if needed**

This ensures accurate metadata for your organized movie library!

See [INTERACTIVE_VERIFICATION.md](INTERACTIVE_VERIFICATION.md) for full documentation.

## Features

- Automatically organize and rename video files
- Interactive movie title/year verification with TMDb & IMDb
- Set audio language metadata for video files
- Support for multiple audio tracks
- Smart filename extraction and cleaning
- Subtitle file detection and organization
